# AI Chat 项目介绍

## 工作流程

在开始工作前，请先：

1. 阅读本文档 (agent.md)
2. 阅读 memory.md 了解项目历史操作
3. 每次修改后都要更新 docs 文档和 memory.md

## 项目概述

一个简单的 AI 聊天应用 Demo，使用现代化前端技术栈构建。

## 技术栈

- **构建工具**: Vite
- **前端框架**: React
- **样式方案**: UnoCSS
- **UI 组件**: Radix UI
- **后端接口**: 通用 OpenAI 兼容接口

## 核心功能

1. 对话界面
2. 消息发送与接收
3. 消息列表展示
4. 响应式布局

## 项目目标

快速搭建一个可运行的 AI 聊天 Demo，展示基础的聊天交互功能。

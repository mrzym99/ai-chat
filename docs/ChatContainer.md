# ChatContainer 组件

## 概述

ChatContainer 组件用于展示所有聊天消息，提供滚动容器和自动滚动功能。

## Props

| 属性 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| messages | Message[] | 是 | - | 消息数组 |
| className | string | 否 | '' | 额外类名 |

## 功能特性

1. **消息列表渲染**
   - 遍历展示所有消息
   - 支持消息间距

2. **自动滚动到底部**
   - 新消息到达时自动滚动
   - 用户手动滚动时暂停自动滚动

3. **空状态**
   - 无消息时显示欢迎提示

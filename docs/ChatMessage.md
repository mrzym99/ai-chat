# ChatMessage 组件

## 概述

ChatMessage 组件用于展示单条聊天消息，支持用户消息和 AI 助手消息两种样式。

## Props

| 属性      | 类型                  | 必填 | 默认值 | 说明     |
| --------- | --------------------- | ---- | ------ | -------- |
| role      | 'user' \| 'assistant' | 是   | -      | 消息角色 |
| content   | string                | 是   | -      | 消息内容 |
| timestamp | Date                  | 否   | -      | 时间戳   |
| avatar    | string                | 否   | -      | 头像 URL |

## 消息角色样式

### 用户消息

- 靠右对齐
- 蓝色背景
- 白色文字

### AI 助手消息

- 靠左对齐
- 灰色背景
- 深色文字
